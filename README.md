# InDesign Recipe Automation 🥘

Скрипт для автоматической верстки рецептов в Adobe InDesign из `.docx` и картинок на macOS.

## Описание работы
Скрипт проверяет папку `input/` на наличие файлов `.docx` (например, `Торт.docx`).
Затем скрипт парсит текст из Word (заголовок = первая строка, остальное = ингредиенты) и **автоматически достает первую добавленную картинку прямо из Word-документа!** Вы также можете просто положить парную картинку в ту же папку `Торт.jpg` (и скрипт возьмет ее).

После этого он автоматически открывает InDesign в фоновом режиме без всплывающих окон.

Скрипт находит внутри шаблона InDesign фреймы, заменяет текст (ориентируясь на стили шрифтов "Title" и "Ingredients/Body") и вставляет нужную картинку в самый большой прямоугольник, применяя автофокус (обрезку по центру).

Затем он сохраняет готовый файл `.indd` и генерирует `.pdf` в папку `output/`.

## 🛠 Установка (Для Вайбкодеров)

### Шаг 1: Права доступа macOS (Очень важно!)
Так как скрипт управляет другим приложением (InDesign) через терминал (AppleScript/osascript), macOS должна дать на это разрешение.
1. Откройте **Системные настройки (System Settings)** Mac.
2. Перейдите в **Конфиденциальность и безопасность (Privacy & Security)**.
3. Откройте **Автоматизация (Automation)**.
4. Разверните `Terminal` (или ту программу, откуда вы запускаете скрипт: `Cursor`, `VSCode`).
5. Убедитесь, что галочка напротив **System Events** и **Adobe InDesign 2024** включена.

### Шаг 2: Установка зависимостей
Скрипту нужен Python (он у вас, скорее всего, уже есть) и маленькая библиотека для чтения Word-файлов. Откройте терминал в папке проекта и выполните:
```bash
pip3 install -r requirements.txt
```

### Шаг 3: Подготовка шаблона
1. Создайте свой файл-шаблон `recipe_template.indd` и положите его в папку `templates/`.
2. В файле должно быть:
   - Текстовый фрейм, в котором к тексту применен `Paragraph Style` с именем `Title` или `Заголовок`. Сюда попадет первая строка из `.docx`.
   - Текстовый фрейм, в котором применен `Paragraph Style` с именем `Ingredients` или `Body`. Сюда попадет остальной текст.
   - Прямоугольник (Rectangle) для картинки (без заливки, просто пустой фрейм для графики). Скрипт сам найдет его (он ищет либо самый большой на странице, либо фрейм с `Script Label` = `Photo`).

## 🚀 Как пользоваться
1. Получите API-ключ от OpenAI (токен, начинающийся с `sk-...`).
2. В терминале задайте его как переменную окружения:
   ```bash
   export OPENAI_API_KEY="ваш_ключ_здесь"
   ```
3. Закиньте файл(ы) в папку `input/`, например:
   - `Pancake.docx` (с текстом и вставленной внутрь картинкой)
   *или* 
   - `Pancake.docx` (с текстом) и `Pancake.jpg`
4. Откройте терминал в папке проекта и запустите:
   ```bash
   python3 build_recipe.py
   ```
3. С запущенным где-то на фоне InDesign произойдет магия.
4. Готовый PDF и INDD файл появятся в папке `output/`.
